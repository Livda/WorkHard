\section{Propriétés à préserver détectées par le groupe D}

Nous rappelons que la catégorie d'utilisateurs ciblée par le groupe D
concerne les Africains.



%%% Dupliquer la "subsection" suivante autant de fois que nécessaire.

\subsection{XXX}  %  XXX = Nom d'une fonctionnalité ou d'un thème ou
                  %  d'un aspect ou...

\begin{enumerate}
\item {\em Bons titres : } Les titres qui sont présent en rouge dans les pages, sont lisibles et explicites.	%  XXX = Nom  d'un aspect, YYY = qqs détails
\item {\em Réseau sociaux : } La présence des boutons pour les réseaux sociaux est pratique.
\item {\em La barre de recherche : } La barre de recherche est pratique pour aller directement là où l'on le souhaite.
\item {\em Menu sur la gauche : } Le menu à gauche est le premier à être vu et lu. Les personnes interviewées l'ont trouvé bien fait.
\item {\em Multi-langues : } Les boutons pour afficher le site en français ou en anglais sont bien visibles. Les étudiants ne parlant pas bien français pourront facilement passer à la version anglaise.
\end{enumerate}
